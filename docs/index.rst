Welcome to the Open GPIAS Documentation
=======================================

Open GPIAS is an open-source tool to record the acoustic startle reflex (ASR) of rodents.

Note
----

If you encounter any bugs or unexpected behaviour, you are encouraged to report a bug in our
Bitbucket `bugtracker <https://github.com/rgerum/open_gpias/issues>`_.
